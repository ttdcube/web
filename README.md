
# ClinicFlow - Hệ thống Đặt Lịch Khám Bệnh Trực Tuyến

## Cách chạy hệ thống

1. Đảm bảo bạn đã cài đặt Python 3.x
2. Nhấp đôi file `start.bat` để chạy server (hoặc chạy lệnh `python -m http.server 8000` trong thư mục gốc)
3. Mở trình duyệt và truy cập: http://localhost:8000/

## Tài khoản thử nghiệm

- **Admin**: `admin@clinicflow.com` / `admin123`
- **Bác sĩ**: `doctor@clinicflow.com` / `123456`
- **Người dùng**: `patient@example.com` / `123456`

## Tính năng hệ thống

### Người dùng
- Trang chủ giới thiệu dịch vụ
- Đăng ký/Đăng nhập
- Đặt lịch khám
- Xem lịch sử cuộc hẹn
- Chỉnh sửa hồ sơ
- Chế độ tối (Dark Mode)

### Bác sĩ
- Dashboard tổng quan
- Quản lý lịch khám
- Danh sách bệnh nhân
- Chỉnh sửa hồ sơ
- Cấu hình lịch làm việc
- Chế độ tối

### Admin
- Quản lý bác sĩ
- Quản lý bệnh nhân
- Quản lý lịch khám
- Dashboard thống kê
- Chế độ tối
