# ClinicFlow - Web đặt lịch khám bệnh

Ứng dụng web quản lý hồ sơ bệnh nhân cơ bản, lịch khám bác sĩ và đặt lịch khám trực tuyến.

## Chức năng

- Đặt lịch khám theo chuyên khoa, bác sĩ, ngày và khung giờ.
- Quản lý danh sách bệnh nhân.
- Quản lý lịch khám với trạng thái chờ xác nhận, đã xác nhận, đã khám và đã hủy.
- Kiểm tra khung giờ còn chỗ.
- Lưu dữ liệu bằng localStorage.
- Xuất lịch khám ra CSV.
- Giao diện sáng/tối và responsive.
